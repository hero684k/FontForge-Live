# 🎨 FontForge Live

Живая песочница шрифтов с эффектами. Просмотр всех системных шрифтов в реальном времени.

---

## Возможности

- Все системные шрифты в одной матрице
- Настройка цвета текста и фона
- Размер шрифта от 14 до 45px
- Жирность, курсив, подчёркивание
- Межсимвольный интервал
- Копирование текста (Ctrl+C)
- Сохранение в PNG
- Тёмный интерфейс

---

## Установка

Скачай `FontForge.exe` из [Releases](https://github.com/hero684k/FontForge-Live/releases)
